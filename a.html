POST request
Method create
Create a new projet. Need to be connected to create one.

Form :

  • title
  • description
  • type

Validate :

(HTTP status_code | detail)
  • 201 : created project

Errors :

(HTTP status_code | detail)
  • 400 : Element doesn't exist | Invalid form
  • 403 : Not permission to create
  • 500 : Intern error
